# **App Name**: TCAM Website

## Core Features:

- Home/About Us: Display an overview of TCAM, its vision, mission, brief faculty highlights, and core values.
- Instruments & Courses: List available instruments and courses with details on duration, fees, and schedule.
- Contact Page: Provide contact details (phone, email, address) with an embedded Google Map and a contact form.
- Enroll Now: A call-to-action page for enrollment with a course selection dropdown, fee information, and next steps.
- Book a Free Demo Session: A scheduling form to book free demo sessions, capturing date/time selections, and storing bookings in Firestore.
- Student Registration: Collect student details (full name, email, phone, instrument/course, preferred batch/time), validate fields, and store in Firestore.
- Inspiration Processing: AI tool extracts style guidelines, structuring, and inspiration details from https://institute.thememusic.in/

## Style Guidelines:

- Primary color: Deep violet (#483D8B) to reflect the creative, musical aspects.
- Background color: Light grey (#E6E6FA) to provide a neutral backdrop.
- Accent color: Light purple (#B098C4) for interactive elements and highlights.
- Body and headline font: 'Inter' sans-serif for a modern, neutral feel.
- Use minimalist icons for navigation and course categories.
- Responsive grid layout to adapt to various screen sizes.
- Subtle transitions and animations for page loads and interactive elements.
